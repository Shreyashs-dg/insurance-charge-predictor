# Insurance Charge Prediction (ML Project)

##  Overview

This project predicts medical insurance charges based on user inputs like age, BMI, smoking habits, and region using Machine Learning.

## Tech Stack

* Python
* Pandas
* Scikit-learn
* Linear Regression

##  Features

* Data preprocessing
* OneHot Encoding using ColumnTransformer
* Linear Regression model
* User input-based prediction system

## Model Performance

* R² Score: ~0.87

##  How to Run

```bash
pip install -r requirements.txt
cd src
python main.py
```

##  Sample Input

* Age: 30
* BMI: 25
* Smoker: No
